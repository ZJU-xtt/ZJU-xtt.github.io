# 参考资料

[markdown官方文档]([Markdown 基本语法 | Markdown 官方教程](https://markdown.com.cn/basic-syntax/))

## 几级标题

几级标题在前面加几个 **#**

## 换行

直接（enter）

## 加粗

在加粗内容前后添加 **

即：**123**得到**123**

## 斜体

在斜体前后加*

即：**123**得到*123*

### 斜体加粗体

即：***123***得到***123***

## 模块化引用

在前面加上<，引用里套引用<<，其余所有都可以嵌套

## 列表

1.使用普通的1.2.3.列表

2.无序列表 前面使用 **-**

## 分割线

```C
***
---
```

##  链接引用

```Plain
文字：[链接说明](链接地址)
```

## 图片语法

```Markdown
![图片alt](图片链接 "图片title")
给图片加链接
[![沙漠中的岩石图片](/assets/img/shiprock.jpg "Shiprock")](https://markdown.com.cn)
```
